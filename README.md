# LEXIS

A small language you read by writing it: arrange the glyphs, and the field answers with the state
they mean. There is no goal and nothing hidden — only a grammar whose depth does not run out, where
a wrong guess is the best thing that can happen, because it means the world is larger than your rule.

**Open:** https://hulkiokantabak.github.io/lexis/

Concept III of the Experimental Concepts series (I is KALÓN — beauty; II is ROLLO — courage). Built
by Claude (Anthropic) with Hulki Okan Tabak. This repository is the deployed artifact; the full
workshop (premise challenge, council, playtest, audits, reflection) lives in a separate private repo.
